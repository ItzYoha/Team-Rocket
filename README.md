🚀 Team Rocket: AI Productivity Planner
An aesthetic, high-performance productivity suite designed for university students. This project leverages Next.js 15, Turborepo, and Gemini AI to help students break down complex academic tasks and maintain deep focus.
🛠 Tech Stack
Framework: Next.js 15 (App Router)
Monorepo Tooling: Turborepo
AI Engine: Google Gemini 1.5 Flash
Styling: Tailwind CSS with Glassmorphism UI

Deployment: Vercel
📦 Monolab-UI Integration
This project features a deep integration with monolab-ui, a custom internal component library designed for consistency across the monorepo.
Shared Design System: Core UI elements like the Navbar, Buttons, and Card containers are managed within the @repo/ui workspace.
Optimized Performance: Components are transpiled directly through the Next.js build pipeline for zero-latency rendering.
Consistency: Ensures that the web and docs applications maintain a unified aesthetic and functional logic.

✨ Key Features
AI Task Breakdown: Input a "Big Task" (like "Study for CAT I") and receive 5 actionable sub-steps generated by Gemini AI.
Advanced Focus Timer: A precise HH:MM:SS timer with custom duration pickers and quick presets (15m, 25m, 50m).
Persistent Task Manager: Robust to-do list with unique ID tracking and local storage synchronization.
Centralized Navigation: A sleek, backdrop-blur navbar for seamless switching between Dashboard, Analytics, and Planner.

🚀 Getting Started
Clone the repo: git clone https://github.com/ItzYoha/Team-Rocket.git
Install dependencies: npm install (from the root directory)
Set up Environment Variables: Add NEXT_PUBLIC_GEMINI_API_KEY to your .env.local
Run development mode: npx turbo run dev
